# Визуализатор графа зависимостей для менеджера пакетов зависимостей (Вариант №15)

`depviz.py` — это утилита для анализа и визуализации зависимостей между пакетами/модулями. Инструмент принимает текстовый файл со списком зависимостей и строит граф в формате PNG, используя Graphviz.

## Формат входного файла

Файл репозитория должен иметь формат вида:

```
A: B C
B: D
C: D E
D:
```

Каждая строка:

* Слева — имя пакета
* После двоеточия — список зависимостей через пробел (можно оставить пустым)

## Пример минимального файла

```
X: Y
Y: Z
Z: X
```

## Требования

* Python 3.8+
* Graphviz установлен в системе и доступен в PATH
* Модуль `graphviz` для Python:

```
pip install graphviz
```

## Использование

```
python depviz.py --package <имя_пакета> --repo <путь_к_файлу> --output <имя_файла> --depth <глубина>
```

### Пример

```
python depviz.py --package X --repo test_repos/repo_cycle.txt --output graph_cycle.png --depth 5
```

## Пояснения

* Если граф содержит циклы, они будут отображены, что позволяет визуально понимать проблемы архитектуры.
* Ограничение глубины (`--depth`) помогает избежать слишком больших графов.
